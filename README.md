# Projet Portfolio John Doe

Pour ce projet nous devons créer le site internet professionel/portfolio de John Doe, étudiant en dev web.
Le site est développé avec **React** et utilise le framework **Bootstrap** pour la mise en page.

## Ce que j'ai utilisé
1. **React JS** 
2. **React Router DOM** 
3. **Bootstrap 5**
4. **FontAwesome**
5. **Google Maps API**

## Structure du projet 
Il est composé de plusieurs pages :
* `Home` qui est la page d'accueil;
* `Services` avec une liste de prestations;
* `Portfolio` avec une galerie photos;
* `Contact` proposant un formulaire de contact;
* `MentionLegales` un accordéon Bootstrap.


## Instructions d'installation
1. Clonez le repository GitHub.
2. Dans le terminal `npm install` pour installer les dépendances.
3. `npm start` pour lancer le serveur de développement.

## Lancement du projet
Exécutez la commande suivante pour lancer l'application en mode développement :
`npm start`.
